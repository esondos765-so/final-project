# Sondos Elsayed Portfolio Website

A modern, responsive portfolio website showcasing frontend development projects and skills.

## 🚀 Live Demo
[View Live Site](https://sondoselsaidsayam-dev.github.io/portfolio/)

## 📋 Project Overview
This is a complete portfolio website built for academic project submission, featuring:
- 4-page responsive website (Home, About, Projects, Contact)
- Modern glassmorphism design with dark/light mode toggle
- Interactive JavaScript features (mobile menu, slider, form validation)
- Bootstrap 5 for responsive layout
- Font Awesome icons

## 🛠️ Technologies Used
- **HTML5** - Semantic markup and structure
- **CSS3** - Custom styling with variables, animations, and responsive design
- **JavaScript (ES6+)** - Interactive features and DOM manipulation
- **Bootstrap 5.3.3** - Responsive grid system and components
- **Font Awesome 6.5.1** - Icons and visual elements

## 📁 Project Structure
```
portfolio/
├── home.html          # Landing page with hero, skills, and project slider
├── about.html         # Personal bio and skills section
├── projects.html      # Portfolio showcase with 6 projects
├── contact.html       # Contact form with validation
├── style.css          # Main stylesheet with themes and animations
├── script.js          # JavaScript for interactions and effects
└── bootstrap-5.0.2-dist/  # Bootstrap framework files
```

## ✨ Features
- **Responsive Design**: Works perfectly on all devices
- **Dark/Light Mode**: Theme toggle with local storage persistence
- **Interactive Slider**: Auto-playing project showcase with manual controls
- **Mobile Menu**: Collapsible navigation for mobile devices
- **Form Validation**: Client-side validation for contact form
- **Smooth Animations**: CSS transitions and JavaScript reveal effects
- **Accessibility**: ARIA labels and semantic HTML

## 🚀 Deployment
This site is deployed on GitHub Pages. To deploy your own version:
1. Fork this repository
2. Go to Settings > Pages
3. Select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Save and wait for deployment
